# Unity Legal Consultancy Website

Static bilingual website prepared for GitHub Pages.

## Publish
1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. In **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/ (root)`.
4. Replace `YOUR-DOMAIN.example` in `CNAME`, `robots.txt`, and `sitemap.xml` with the actual domain.
5. Configure the DNS records shown by GitHub Pages and enable **Enforce HTTPS** after the certificate is issued.

## Update content
- Arabic pages are in `/ar`.
- English pages are in `/en`.
- Article templates are in `/ar/insights/article-template.html` and `/en/insights/article-template.html`.
- PDF reports can be added to `/assets/reports`.
- Main styling is in `/assets/css/styles.css`.

## Email
All consultancy buttons use `unitylegalconsultancy@outlook.com` and open a prepared, editable message in the visitor's default email application.
