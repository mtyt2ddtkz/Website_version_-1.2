# Pre-launch checklist

- Replace `YOUR-DOMAIN.example` in CNAME, robots.txt and sitemap.xml.
- Confirm the final Arabic and English company names.
- Replace the temporary U monogram with the approved logo files, if available.
- Test every navigation link in Arabic and English.
- Test email links on macOS, iPhone and Windows.
- Review all service descriptions and disclaimer text.
- Confirm GitHub Pages HTTPS is enabled.
- Check the site on Safari, Chrome and mobile screens.
