# دليل إضافة مقال أو تقرير

1. انسخ الملف `ar/insights/article-template.html`.
2. أعد تسمية النسخة باسم إنجليزي قصير، مثل `contract-review.html`.
3. عدّل العنوان والتاريخ والتصنيف والنص.
4. أضف بطاقة للمقال داخل `ar/insights/index.html`.
5. كرر الخطوات للنسخة الإنجليزية داخل `en/insights`.
6. لإرفاق PDF، ضعه في `assets/reports` وأضف رابطاً إليه داخل المقال.
